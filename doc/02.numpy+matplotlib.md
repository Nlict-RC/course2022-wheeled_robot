## json

一种轻量级资料交换格式。其内容由属性和值所组成，因此也有易于阅读和处理的优势。

```python
## sender
vel = {
    "x":0.0, # m/s
    "w":0.0, # rad/s
}
data = json.dumps(vel)
print(data)
sock.sendto(data.encode('utf-8'), _ENDPOINT)

## receiver
data, addr = sock.recvfrom(65535)
data = data.decode()
vel = json.loads(data)
print(vel["x"],vel["w"])
```
